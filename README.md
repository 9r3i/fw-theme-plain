# fw-theme-plain
Real Plain -- ForceWebsite Theme


# Config
For theme config simply put this in theme section
```json
{
  "theme": {
    "namespace": "plain",
    "host": "https://9r3i.github.io/fw-theme-plain",
    "config": {
      "data": {
        "limit": 8
      }
    }
  }
}
```
